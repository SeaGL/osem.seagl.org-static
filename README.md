# Static archive of OSEM

This is an archive of the publicly browsable portion of [SeaGL’s OSEM instance], so that the aging backend may be decommissioned while the front stays [cool].

Repository contents:

- [`archiver`](/archiver/): Script for performing archival and export
- [`archive`](/archive/): Raw archive data
- [`github-pages`](/github-pages/): Export as a static site compatible with GitHub Pages

[cool]: https://www.w3.org/Provider/Style/URI
[SeaGL’s OSEM instance]: https://osem.seagl.org/
